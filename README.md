# Fraud Detection Suite

A machine learning-based fraud detection system that identifies:
- 📧 Phishing emails
- 💳 Credit card fraud transactions

The project combines NLP and tabular machine learning models into a single interactive Streamlit application.

---

## Overview

This project demonstrates how machine learning can be applied to detect fraudulent activity in different domains:

- Email phishing detection using text classification
- Credit card fraud detection using transaction data
- Real-time prediction through a web interface

---

## Models Used

- Logistic Regression
- Random Forest
- XGBoost (primary model)

---

## Features

- Real-time fraud prediction
- Probability-based risk scoring
- Simple Streamlit UI for interaction
- Supports both text and numerical inputs

---

## Tech Stack

- Python
- Scikit-learn
- XGBoost
- Pandas / NumPy
- Streamlit
- Joblib
